#### Text Attributes
- Reset All Attributes: `\033[0m`
- Set Bold: `\033[1m`
- Set Dim: `\033[2m`
- Set Italic: `\033[3m`
- Set Underline: `\033[4m`
- Set Blink: `\033[5m`
- Set Inverse: `\033[7m`
- Set Hidden: `\033[8m`
- Set Strikethrough: `\033[9m`
- Set Foreground Color: `\033[38;5;#m` (# is color code)
- Set Background Color: `\033[48;5;#m` (# is color code)
